# ## 1. What is Firebase?

Firebase is a **Google-backed platform** providing cloud tools for building web and mobile applications.  
It includes services such as:

- Authentication
    
- Realtime Database
    
- Firestore
    
- Hosting
    
- Cloud Storage
    
- Cloud Functions
    
- Analytics
    

This PPT focuses on **Authentication + Client SDK + Firebase CLI**.

---

# ## 2. Firebase Console

The Firebase Console is the **dashboard** where you manage your Firebase project.

### With Firebase Console you can:

- Create projects
    
- Enable authentication providers
    
- View analytics
    
- Manage Firestore / Storage
    
- Configure hosting & cloud functions
    

---

# ## 3. Setup Sample Application in Firebase

To use Firebase in your application:

1. Create a Firebase Project
    
2. Add a Web App inside the project
    
3. Copy the Firebase SDK configuration
    
4. Install & initialize Firebase in your code
    
5. Enable Authentication (Email/Password, Google, Phone, etc.)
    

---

# ## 4. Firebase JavaScript Client SDK

### **Installation**

```
npm install firebase
```

### **Initialization Code Example**

(_Sensitive data removed - do not expose API keys._)

```javascript
import { initializeApp } from "firebase/app";
import { getAnalytics } from "firebase/analytics";

// Firebase configuration (example)
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "yourapp.firebaseapp.com",
  projectId: "yourapp",
  storageBucket: "yourapp.appspot.com",
  measurementId: "G-XXXXXXX"
};

// Initialize Firebase
const app = initializeApp(firebaseConfig);
const analytics = getAnalytics(app);
```

### Key Notes

- The SDK imports only required Firebase services.
    
- Measurement ID is optional.
    
- You must **remove unnecessary comments** from boilerplate code.
    
- Keep Firebase API keys secure and avoid hardcoding in public repos.
    

---

# ## 5. Firebase Authentication Integration (Overview)

To integrate authentication:

1. Enable authentication provider in **Firebase Console**.
    
2. Install Firebase SDK.
    
3. Initialize the Firebase app.
    
4. Use auth functions such as:
    
    - `createUserWithEmailAndPassword()`
        
    - `signInWithEmailAndPassword()`
        
    - `onAuthStateChanged()`
        
5. Secure backend routes using Firebase tokens.
    

Firebase Authentication supports:

- Email/Password
    
- Google
    
- GitHub
    
- Facebook
    
- Phone
    
- Anonymous auth
    

---

# ## 6. Firebase CLI

The **Firebase CLI** is used for deploying and managing Firebase services via terminal.

### **Basic Command Structure**

```
firebase <command> [options] [arguments]
firebase --help
```

### **Installation**

```
npm install -g firebase-tools
```

### **Authentication**

```
firebase login
```

### **Create a Firebase Project**

```
firebase projects:create
```

### Firebase CLI Documentation

[https://firebase.google.com/docs/cli](https://firebase.google.com/docs/cli)

---

# ## 7. Typical Workflow Using Firebase CLI

1. Install Firebase CLI
    
2. Login to your Google account
    
3. Initialize your project:
    
    ```
    firebase init
    ```
    
4. Select features:
    
    - Hosting
        
    - Firestore
        
    - Functions
        
    - Storage
        
5. Deploy:
    
    ```
    firebase deploy
    ```
