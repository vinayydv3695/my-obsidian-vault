### **Download Link**

[https://www.mongodb.com/try/download/community](https://www.mongodb.com/try/download/community)

### **Steps**

1. Download the **MongoDB Community Server** installer.
    
2. Install it with default settings.
    
3. It also installs **MongoDB Compass**, the official GUI tool.
    
4. You may need to manually start the MongoDB server using:
    
    ```
    mongod
    ```
    
5. MongoDB requires a data directory:
    
    - Windows default: `C:\data\db`
        
    - Linux/Mac example: `/data/db`
        

If the directory does not exist, manually create it.

# ## 2. Installing Mongo Shell (mongosh)

### **Download Link**

[https://www.mongodb.com/try/download/shell](https://www.mongodb.com/try/download/shell)

### **Steps**

1. Download **mongosh** installer.
    
2. Install and add its path to **Environment Variables** so that `mongosh` can run globally from terminal.
    
3. Verify installation:
    
    ```
    mongosh --version
    ```

# ## 3. Installing Node.js and NPM

### **Download Link**

[https://nodejs.org/en/download](https://nodejs.org/en/download)

### **What gets installed?**

- Node.js runtime
    
- npm (Node Package Manager)
    

### **Verify installation**

Open CMD or terminal and run:

```
node -v
npm -v
```

This confirms the installation was successful.
