

## 1. Introduction to CSS

- **CSS** stands for **Cascading Style Sheets**
    
- CSS describes **how HTML elements are displayed on screen**
    
- It improves:
    
    - Presentation
        
    - Readability
        
    - Reusability
        
- CSS makes web design **time-efficient**
    
- CSS code is written inside:
    

```html
<style> </style>
```

---

## 2. Ways to Apply CSS

CSS can be applied in **three ways**:

1. Inline CSS
    
2. Internal CSS
    
3. External CSS
    

---

## 3. Inline CSS

- Used to apply style to **a single element**
    
- Written inside the HTML tag using `style` attribute
    
- Has **highest priority**
    

Example:

```html
<p style="color:red; font-size:20px;">Hello</p>
```

‚úÖ Used for unique styling  
‚ùå Not recommended for large projects

---

## 4. Internal CSS

- CSS written inside `<style>` tag
    
- Placed inside `<head>` section
    
- Used when **only one page** needs unique styling
    

Example:

```html
<head>
  <style>
    p { color: blue; }
  </style>
</head>
```

---

## 5. External CSS

- CSS written in a separate `.css` file
    
- Linked using `<link>` tag inside `<head>`
    
- One CSS file can style **entire website**
    

HTML file:

```html
<link rel="stylesheet" href="style.css">
```

CSS file:

```css
p {
  color: green;
}
```

‚úÖ Best practice  
‚úÖ Easy maintenance

---

## 6. CSS Selectors

CSS selectors are used to **select HTML elements to style**

### Types of Selectors

- Element Selector
    
- Class Selector
    
- ID Selector
    

---

## 7. Element Selector

- Selects elements by **tag name**
    

CSS:

```css
p {
  color: red;
}
```

Applies to **all `<p>` elements**

---

## 8. Class Selector

- Selects elements with a specific **class**
    
- Uses dot (`.`)
    
- Can be applied to **multiple elements**
    

HTML:

```html
<p class="text">Hello</p>
```

CSS:

```css
.text {
  color: blue;
}
```

---

## 9. ID Selector

- Selects element with specific **id**
    
- Uses hash (`#`)
    
- **ID must be unique**
    

HTML:

```html
<p id="para1">Hello</p>
```

CSS:

```css
#para1 {
  color: green;
}
```

---

## 10. Background Styling

### Background Properties

- `background-color`
    
- `background-image`
    
- `background-repeat`
    
- `background-attachment`
    
- `background-position`
    
- `background-size`
    

Example:

```css
body {
  background-color: lightblue;
}
```

---

## 11. Margin Styling

- Margin creates **space outside the border**
    
- Individual sides:
    
    - margin-top
        
    - margin-right
        
    - margin-bottom
        
    - margin-left
        

Shorthand:

```css
margin: 25px 50px 75px 100px;
```

---

## 12. Padding Styling

- Padding creates **space inside the border**
    
- Affects content spacing
    

Difference:

- Margin ‚Üí outside
    
- Padding ‚Üí inside
    

---

## 13. CSS Box Model

The box model consists of:

1. Content
    
2. Padding
    
3. Border
    
4. Margin
    

Order:

```
Margin ‚Üí Border ‚Üí Padding ‚Üí Content
```

---

## 14. Position Property

### Types of Positioning

- `static` (default)
    
- `relative` ‚Üí relative to normal position
    
- `absolute` ‚Üí relative to nearest positioned parent
    
- `fixed` ‚Üí relative to viewport
    
- `sticky` ‚Üí based on scroll position
    

Example:

```css
div {
  position: fixed;
  top: 0;
}
```

---

## 15. Overflow Property

Controls content overflow

Values:

- visible (default)
    
- hidden
    
- scroll
    
- auto
    

Example:

```css
div {
  overflow: scroll;
}
```

---

## 16. Float Property

Used for positioning elements

Values:

- left
    
- right
    
- none
    
- inherit
    

Example:

```css
img {
  float: left;
}
```

---

## 17. Display Property

Controls element display behavior

Common values:

- block
    
- inline
    
- inline-block
    
- none
    

Example:

```css
span {
  display: block;
}
```

# üìò CSS3 ‚Äì Detailed Notes 

## 1. Introduction to CSS3

- **CSS3** is the newer version of CSS used for advanced web styling
    
- CSS3 is divided into **modules**
    
- Makes styling:
    
    - Easier
        
    - More organized
        
    - More powerful
        
- Enables:
    
    - Dynamic pages without JavaScript
        
    - Responsive design
        
- Introduces advanced selectors, animations, transformations, and layouts
    

---

## 2. Features & Applications of CSS3

- Advanced Selectors
    
- Dynamic Selectors
    
- Enhanced Box Model
    
- Color Models with opacity
    
- Transformations
    
- Perspective (3D)
    
- Animations
    
- Flexbox layout
    

---

## 3. Advanced Selectors

### Types of Advanced Selectors

- Universal Selector
    
- Child Selector
    
- Adjacent Sibling Selector
    
- General Sibling Selector
    
- Attribute / Relational Selectors
    

---

### Universal Selector (`*`)

- Selects **all elements**
    
- Often used to reset margins and paddings
    

```css
* {
  margin: 0px;
}
```

---

### Child Selector (`>`)

- Selects **direct children only**
    
- Syntax:
    

```css
parent > child
```

Example:

```css
ul > li {
  color: red;
}
```

---

### Adjacent Sibling Selector (`+`)

- Selects element immediately following another
    
- Syntax:
    

```css
E1 + E2
```

Example:

```css
h1 + p {
  color: blue;
}
```

---

### General Sibling Selector (`~`)

- Selects **all sibling elements** after the first
    
- Syntax:
    

```css
E1 ~ E2
```

Example:

```css
h1 ~ p {
  color: green;
}
```

---

## 4. Dynamic Selector (Show / Hide)

- Uses **radio buttons + sibling selectors**
    
- Allows showing/hiding elements **without JavaScript**
    
- Relies on:
    
    - `:checked`
        
    - `+` or `~` selectors
        

---

## 5. CSS3 Box Model

- CSS box model contains:
    
    - Content
        
    - Padding
        
    - Border
        
    - Margin
        

### Default Calculation

```
Width + Padding + Border = Total Width
Height + Padding + Border = Total Height
```

---

## 6. Box Sizing Property

- Solves layout issues in responsive design
    
- Makes width/height include padding and border
    

### `box-sizing: border-box`

```css
* {
  box-sizing: border-box;
}
```

- Width includes:
    
    - Content
        
    - Padding
        
    - Border
        

---

## 7. CSS3 Color Models

### Opacity Property

- Controls transparency of entire element
    
- Default value: `1`
    
- Range: `0` (transparent) to `1` (opaque)
    

```css
div {
  background-color: red;
  opacity: 0.5;
}
```

‚ö† Affects **child elements also**

---

### RGBA Color Model

- RGBA = Red, Green, Blue, Alpha
    
- Alpha controls **only color opacity**
    

```css
background-color: rgba(255, 0, 0, 0.5);
```

‚úÖ Text remains unaffected  
‚úÖ Preferred over `opacity`

---

## 8. CSS3 Transformations

Used to change element appearance

### Transform Functions

- translate()
    
- scale()
    
- rotate()
    
- skew()
    

---

### translate()

```css
transform: translate(15px, -15px);
```

- Moves element on X and Y axis
    
- Variants:
    
    - translateX()
        
    - translateY()
        

---

### scale()

```css
transform: scale(1.5, 2);
```

- Scales width and height
    
- Single value ‚Üí proportional scaling
    

---

### rotate()

```css
transform: rotate(15deg);
```

- Rotates element clockwise
    

---

### skew()

```css
transform: skewX(15deg);
```

- Skews element on X/Y axis
    

---

## 9. Perspective (3D Effect)

- Adds depth (3D space)
    
- Affects **child elements**, not parent
    
- Lower value ‚Üí stronger 3D effect
    

```css
perspective: 500px;
```

---

## 10. CSS3 Animation

- Enables animation **without JavaScript**
    
- Two parts:
    
    - Animation properties
        
    - `@keyframes`
        

Example:

```css
@keyframes move {
  from { left: 0; }
  to { left: 100px; }
}
```

---

## 11. Flexbox Layout

- Flexible Box layout
    
- Efficient layout, alignment, spacing
    
- Automatically adjusts items
    

### Benefits

- Responsive
    
- Dynamic sizing
    
- Alignment control
    

```css
.container {
  display: flex;
}
```
