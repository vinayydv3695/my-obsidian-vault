

## 1. Introduction to Bootstrap

- **Bootstrap** is a front-end CSS framework
    
- Developed at **Twitter Inc.**
    
- Created by **Mark Otto** and **Jacob Thornton**
    
- Used to design **responsive web pages**
    
- Solves CSS problems such as:
    
    - Browser compatibility
        
    - Responsiveness
        
    - Maintainability of large applications
        
- Provides:
    
    - Grid system
        
    - UI components
        
    - Typography classes
        
    - Forms, tables, navbars, carousel etc.
        

---

## 2. Browser Compatibility

- Different browsers may render CSS differently
    
- Same code can appear different in:
    
    - Chrome
        
    - Internet Explorer
        
- Bootstrap provides **consistent UI across browsers**
    

---

## 3. Responsive Web Design

- Responsive design provides **optimal viewing experience**
    
- Webpage adapts to:
    
    - Mobile
        
    - Tablet
        
    - Desktop
        
- Layout automatically adjusts screen size
    

---

## 4. Getting Started with Bootstrap

Steps:

1. Visit **getbootstrap.com**
    
2. Copy `bootstrap.min.css`
    
3. Paste the `<link>` inside `<head>`
    

Example:

```html
<link rel="stylesheet" href="bootstrap.min.css">
```

---

## 5. Page Layout ‚Äì Container Classes

Bootstrap needs a **container** to wrap page content.

### Types of Containers

- `.container`
    
    - Fixed-width responsive container
        
- `.container-fluid`
    
    - Full-width container (100% viewport)
        

Example:

```html
<div class="container"></div>
<div class="container-fluid"></div>
```

---

## 6. Grid System

- Bootstrap grid system divides page into:
    
    - **Rows**
        
    - **Columns**
        
- Each row contains **12 columns**
    

---

## 7. Column Classes

Syntax:

```html
<div class="col-<size>-<n>"></div>
```

Where:

- `<size>` ‚Üí screen size
    
- `<n>` ‚Üí number of columns (0‚Äì12)
    

### Size Qualifiers

- `lg` ‚Üí ‚â•1200px
    
- `md` ‚Üí ‚â•992px
    
- `sm` ‚Üí ‚â•768px
    
- `xs` ‚Üí <768px
    

---

## 8. Grid Examples

Equal columns:

```html
<div class="col-md-4"></div>
<div class="col-md-4"></div>
<div class="col-md-4"></div>
```

Overflow example:

```html
<div class="col-md-6"></div>
<div class="col-md-3"></div>
<div class="col-md-4"></div>
```

(6 + 3 + 4 = 13 ‚Üí overflow)

---

## 9. UI Components ‚Äì Navigation Bar

- Bootstrap provides a **responsive navbar**
    
- Uses HTML5 `<nav>` element
    
- Automatically collapses on small screens
    
- Displays **hamburger menu**
    

To enable responsiveness:

- `bootstrap.css`
    
- `bootstrap.js`
    
- `jquery.js`
    

---

## 10. Navbar Structure

Includes:

- Brand name
    
- Toggle button
    
- Navigation links
    
- Left & right aligned links
    

---

## 11. Toggle Button (Responsive Navbar)

- Toggle button switches navbar states
    
- Activated on smaller screens
    
- Controlled using Bootstrap JS
    

---

## 12. Carousel Component

- Used to create **slideshow**
    
- Cycles through elements
    
- Used for:
    
    - Banners
        
    - Image sliders
        

---

## 13. Modal & Forms

- Bootstrap supports:
    
    - Modal dialogs
        
    - Button-triggered forms
        
- Example:
    
    - Login form using modal
        

---

## 14. Advantages of Bootstrap

- Responsive by default
    
- Cross-browser compatibility
    
- Reusable components
    
- Faster development
    
- Excellent documentation
