

## 1. Introduction to JavaScript

- **JavaScript** is a **scripting language** used in web browsers
    
- It adds **dynamic and interactive behavior** to web pages
    
- JavaScript works with:
    
    - **HTML** â†’ structure & content
        
    - **CSS** â†’ layout & styling
        
    - **JavaScript** â†’ behavior & interaction
        
- JavaScript is a **client-side language**
    
    - Code executes in the **userâ€™s browser**
        

---

## 2. Why JavaScript?

- HTML â†’ static content
    
- CSS â†’ appearance and layout
    
- JavaScript â†’ dynamic actions such as:
    
    - Changing HTML content
        
    - Modifying attributes
        
    - Changing CSS styles
        
    - Showing / hiding elements
        
    - Handling user events
        

---

## 3. Features of JavaScript

- Client-side scripting language
    
- **Loosely typed**
    
- **Function-oriented**
    
- Supports **DOM manipulation**
    
- Dynamic typing (same variable â†’ different data types)
    

---

## 4. Client-Side Scripting

- JavaScript code is embedded inside HTML
    
- Script is sent to browser along with HTML
    
- Browser executes JavaScript on the client machine
    

Example:

```html
<script>
  alert("Hello");
</script>
```

---

## 5. Loosely Typed Language

- Variables do **not require type declaration**
    
- Type is decided at runtime
    

Example:

```js
var a;
a = 100;      // number
a = true;     // boolean
a = "Hello";  // string
```

- If variable is declared but not assigned â†’ `undefined`
    

---

## 6. Function Oriented Language

- Functions are **first-class objects**
    
- A function can:
    
    - Be passed as argument
        
    - Be returned from another function
        
    - Be nested
        
- Classes are also treated as functions
    

```js
function test() {
  return function () {
    return "Hello";
  }
}
```

---

## 7. JavaScript and DOM

- DOM â†’ **Document Object Model**
    
- Browser converts HTML into a **tree structure**
    
- JavaScript uses DOM to:
    
    - Access elements
        
    - Modify content, attributes, styles
        
    - Add / remove elements
        
    - Handle events
        

---

## 8. Document Object Model (DOM)

- DOM is a **W3C standard**
    
- Platform and language neutral
    
- DOM types:
    
    - **Core DOM**
        
    - **XML DOM**
        
    - **HTML DOM**
        

---

## 9. HTML DOM

HTML DOM defines:

- HTML elements as objects
    
- Properties of elements
    
- Methods to access elements
    
- Events for elements
    

Allows:

- Get, change, add, delete HTML elements
    

---

- Standard: **ECMA-262**
    
- First version released in **1997**
    
- Latest versions include:

    - New RegExp features
        
    - Rest / spread parameters

## 1. Arrays in JavaScript

- Indexing:

## 2. Important Array Properties & Methods

### `length`

- Returns number of elements
    

```js

```

### `push()`

- Adds element(s) at the **end**
    

```js

```

### `unshift()`

- Adds element(s) at the **beginning**
    

### `pop()`

- Removes last element
    

### `shift()`

- Removes first element
    

### `slice(begin, end)`

- Returns a **copy** of part of array
    
- Does **not** modify original array
    

### `splice(index, howmany, itemsâ€¦)`

- Adds / removes elements
    
- **Modifies original array**
